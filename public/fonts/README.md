Place local font binaries here for offline sovereign runtime.

Expected files referenced by `src/templates/css/terminal.css`:

- `Inter-Regular.woff2`
- `Inter-SemiBold.woff2`
- `JetBrainsMono-Regular.woff2`
- `JetBrainsMono-SemiBold.woff2`

If these files are absent, UI will gracefully fall back to system sans/monospace fonts.
